Intructions to start the webapp:

1.Install Node.js.

2.Start command Prompt or any terminal and make sure that the folder containing the files is open in the terminal.Use cd to navigate to said folder.
it will probably look like this:

C:\Users\(Username)\Downloads\ThrustCalculator>

Directory names may change based on where the folder is placed.

3.Type npm start to start the app.

4.Use Ctrl+C to stop the app.

A calculator which fixes the values of static or dynamic thrust to a desired ammount to find Pitch or RPM or Diameter or Vac required for that ammount can be made but would require some time.
